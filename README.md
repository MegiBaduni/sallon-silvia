# Silvia Salon & Spa 💅

Official website for **Silvia Salon & Spa** by Silvia Goxha, a luxury women-only salon based in Sarandë, Albania.

## Live Site
[View Website]

## Features
- Video-first luxury design with real salon footage
- Full treatment showcase — Hair, Nails, Lashes, Makeup, Spa & more
- Online booking form connected to Formspree
- WhatsApp & Instagram direct contact
- Client reviews from across the world
- Google Maps location embed
- Fully mobile responsive
- No frameworks — pure HTML, CSS & JavaScript
